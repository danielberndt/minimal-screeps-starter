# Simple Screeps Starter Script

## Setup

- `git clone [thisrepo]`
- `cd [reponame]`
- `npm install`
- create `.env` file at project root (see below)
- `npm run deploy`

- optionally `rm -rf .git/`

## .env -file

Looks like this:

```
EMAIL="[YOUR_EMAIL]"
PASSWORD="[PASSWORD]"
HOST="[SERVER.IP]"
```
